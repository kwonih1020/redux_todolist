## 라이브러리 🛠
- React.js
- Redux
- Styled Components
- Router

## 환경설정
```
- npm install --global yarn (yarn을 설치 안했을때)
- yarn create react-app "프로젝트 명"
- yarn add styled-components
- yarn add react-redux
- yarn add react-router=dom@6
- yarn start
```
